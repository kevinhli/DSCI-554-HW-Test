3. API integration model
- The dashboard is not making live browser calls to FRED or Yahoo.
- Instead, the APIs are integrated into the optional Python data-build step:
  - `python data/build_dashboard_data.py --fred-api-key <your_key>`
  - or `FRED_API_KEY=<your_key>` and then `python data/build_dashboard_data.py`
- FRED series:
  - always fetched from the official `series/observations` API during the build
  - uses the API key from `--fred-api-key` or `FRED_API_KEY`
- Yahoo series:
  - always fetched during the build through the `yfinance` Python library
- FOMC events:
  - kept as a static reference file in `data/fomc_events.csv`
  - this preserves the full meeting list, including hold events, which is not cleanly available from the market-data APIs alone
- After fetching, the script regenerates the static dashboard JSON used by GitHub Pages.
